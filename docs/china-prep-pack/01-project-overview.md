# Project Overview

## Project name
China Prep Pack

## One-line description
China Prep Pack is a personalized pre-departure travel prep tool for first-time international travelers visiting China.

## Problem
First-time travelers to China often struggle with fragmented, outdated, or non-personalized travel information. They do not just need a guide. They need a practical action pack that helps them prepare for payments, internet access, transport, communication, food restrictions, and first-day adaptation.

## Target user
The MVP focuses on first-time travelers to China who use English, have limited or no Chinese ability, and prefer free or semi-independent travel.

## Core value
The product helps users:

- know what to prepare before departure
- understand what matters most for their trip
- arrive with more confidence
- handle common travel situations with less stress

## MVP output
The first version should generate:

- a priority checklist
- a first-day arrival guide
- bilingual communication cards
- personalized risk reminders

## Not in MVP
The first version does not include:

- flight or hotel booking
- real-time maps
- community content
- live local recommendations
- a full itinerary planner

## Product principle
The system should work in four steps:

1. collect traveler profile data
2. filter and prioritize prep tasks
3. assemble a structured prep pack
4. use AI to generate summary wording and supporting text

## Why this is a good solo-founder project
- narrow wedge
- clear value
- strong user anxiety relief
- manageable MVP scope
- possible to build with rules + AI before heavy data integrations
